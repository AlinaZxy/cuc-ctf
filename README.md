# cuc-ctf
中国传媒大学 CTF 竞赛组织

本仓库用户收集 2020 秋季 CTF 线下培训课后题的 writeup，请同学们按照以下要求提交：

- fork 本仓库。
- 在 `writeups/` 文件夹下，以个人的 ID 建立一个自己的文件夹，个人的 writeup 都存储在自己的文件夹下。
-  向主仓库发送 `PR`。

例：同学 apple 在  `writeups/` 目录下建立了自己的文件夹  `writeups/apple` ，则他个人仓库下的文件目录大致如下：

```
└─ writeups
    └─ apple
        |─ writeup1.md
        |─ writeup2.md
        └─ writeup3.md
```
